# ReneTelegramBot
Librería de utilidades de python para facilitar la creación de boots tipo menu interactivos 
Destacar la habilidad de agregar botones directamente en el teclado con una acción fácil de relacionar 

pip install ReneTelegramBot


ReneTelegramBot es una librería desarrollada en Python que brinda una variedad de métodos de utilidad para la creación y manipulación de un bot de telegram. Esta herramienta ofrece la posibilidad de crear menús interactivos con teclado en el chat y gestionar fácilmente los eventos que ocurren al pulsar los botones, además de manejar la visibilidad de los componentes según las necesidades del desarrollador.

Esta librería posee clases claramente diferenciadas para la manipulación de cada componente del bot de Telegram, lo que favorece una programación estructurada, de fácil uso y manipulación para el desarrollador. Con esta herramienta, los programadores pueden crear chatbots en Telegram con funcionalidades y características avanzadas en cuestión de minutos.
